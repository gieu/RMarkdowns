Hablemos de Distribuciones Muestrales…

Ya sabes cómo cargar los datos y ver un histograma. En el siguiente ejemplo, veremos cómo podemos tomar una muestra de un conjunto grande de datos, y hacer un histograma de esta muestra. Dicha actividad la podemos repetir muchas veces y ver cómo se distribuyen los datos de cada muestra.

Modifica el programa para generar más muestras y muestras más grandes. Cómo se distribuyen los datos de cada muestra?

# Generamos un conjunto de datos de ejemplo que siguen una distribucion uniforme
data <- runif(n=1000, min=1.2, max=6)
# Creamos el histograma
hist(data, col='steelblue', main='Histograma Datos Ejemplo')

# Vamos a sacar 10 muestras de 10 datos
sample <- c()
n<-10 # numero de muestras

for (i in 1:n){
  # La funcion sample me permite sacar muestras aleatorias de un conjunto de datos
  datosMuestra<-sample(data, 30, replace=TRUE) 
  # Generamos el histograma con los datos de la muestra
  hist(datosMuestra, col='steelblue', main='Histograma Datos Ejemplo')
  # Guardamos la media de la muestra para luego graficarla.
  sample[i] = mean(datosMuestra)
}
# Histograma de las medias generadas
hist(sample, col ='red', xlab='Rango Datos', main='Histograma Medias Muestrales')

Ahora cambiemos la distribución original.

En lugar de usar runif, usamos rnorm para que nos genere unos datos de distribución normal.

# Generamos un conjunto de datos de muestra
data <- rnorm(n=1000)
# Creamos el histograma
hist(data, col='steelblue', main='Histograma Datos Ejemplo')

# Vamos a sacar 10 muestras de 30 datos
sample <- c()
n<-10 # numero de muestras

for (i in 1:n){
  # La funcion sample me permite sacar muestras aleatorias de un conjunto de datos
  datosMuestra<-sample(data, 30, replace=TRUE) 
  # Generamos el histograma con los datos de la muestra
  hist(datosMuestra, col='steelblue', main='Histograma Datos Ejemplo')
  # Guardamos la media de la muestra para luego graficarla.
  sample[i] = mean(datosMuestra)
}
# Histograma de las medias generadas
hist(sample, col ='red', xlab='Rango Datos', main='Histograma Medias Muestrales')

Medidas de estadística descriptiva por grupos

Nota que en los datos tenemos una columa que describe el tipo y número de cursos previos: PriorCourses. Si queremos comparar el campo de Pretest a Posttest seún diferentes experiencias, podemos comenzar por ver la media y la desviación estándar para cada grupo utilizando la función aggregate

aggregate(LearningPre~PriorCourses,desempenio,mean)
aggregate(LearningPre~PriorCourses,desempenio,sd)
