Learning outcomes

Al finalizar esta actividad, estarás en capacidad de:

  • Crear variables, asignar y leer valores hacia/desde dichas variables

  • Hacer cálculos simples entre variables/valores

  • Convertir el tipo de dato de las variables.

  • Crear e invocar funciones que hagan cálculos sencillos

Variables

El símbolo <- asigna un valor a una variable Por ejemplo, si queremos asignar el valor 5 a la variable ‘y’, hacemos:

y <- 5
# Let's make sure it has the corresponding value - please execute the following line
y
y <- 5
Ahora podemos usar la variable y. En el siguiente bloque, multiplicamos y por 2, y luego lo asignamos a la variable x. 
y*2
# Since we did not assign it to any variable, 'y' will keep the same value
y
# Now, if we want to save the result in a variable 'x':
x <- y*2
# Check out the value of variable 'x' now
x
y <- 5
x <- y*2

Actividad

  • Divide ‘x’ entre ‘y’ y asigna el resultado a la variable a

  • Verifica que a tiene el valor correcto

Ahora vamos a trabajar con otro tipo de variables. Vamos a crear un listado de datos llamado vector. Un vector es una lista de datos almacenada en una única variable. Para crear un vector, usamos la instrucción c(): 
z <- c(1.1, 9, pi, 5 , 6, 7, 8) 

# What is the meaning of having 'pi' in this vector?

# Check out the value of the variable 'z' 
z
Este vector puede ser usados para crear otros más grandes. Por ejemplo, si queremos crear un vector con todos los valores en z, pero que también incluya los valores ‘x’ y ‘y’, hacemos lo siguiente: 
y <- 5
x <- y*2
z <- c(1.1, 9, pi, 5 , 6, 7, 8) 
c(x,z,y)

Actividad

  • Qué ocurre si multiplicamos un vector por un número? z * 2 + 100

Help - Ayuda

Cada que tengas una pregunta sobre una función de R, puedes acceder a you can access its information using ‘?’ Por ejemplo, si quieres saber qué hace la función mean:

?mean

Preguta

Qué hace la función mean?

Puedes usarla con el vector z, asuí:

mean(z)

Podemos también hacer opereaciones entre vectores. Por ejemplo::

# The following instruction will multiply one by one the elements from both vectors
z * c(2, 1, 2,3,5,6,4)

# When the length is different for both vectors, R 'recycles' the numbers from the smaller vector
# This recylcing process re-uses the numbers on the same order. For instance:
c(2, 2, 4, 4) + c(1, 2)

# The outcome is a vector including four elements, just like executing:
# c(2, 2, 4, 4) + c(1, 2, 1, 2)

Matrices

Ahora, si queremos tener varias columnas en una sola variable, podemos usar una matriz o un data.frame. La principal diferencia entre estas dos estructuras de datos es que las matrices solo pueden almacenar un tipo de datos (ej. números), mientras que los data.frames puede incluir diferentes tipos (ej. números y texto)

# Let's first create a vector called mi_vector containing all integers from 1 to 20.
# We can do that using the operator ':' 
mi_vector <-1:20
# Check out the content of mi_vector
mi_vector

# To validate the data type, we can use the function 'class'
class(mi_vector)

# If we want to know how many elements are in there, we can use the function 'length'
length(mi_vector)

# Let's use an appropriate name for our variable

Ahora imagina que los números de cada fila corresponden a estudiantes, de los cuales contamos con cinco valores (columnas): edad, altura, grado, promedio y peso (age, height, grade, GPA, and weight).

Nos gustaría saber qué estudiante corresponde a cada fila, así que creamos una columna adicional

students <- c('Charlie','Hayden','Alex','Ben')
mi_data <- cbind (students, mi_matrix)

# Now all the values in mi_matrix have quotation marks around them
mi_data

Como ves, ahora tenemos comillas alrededor de cada dato que tenemos. Esto ocurre porque una matriz puede solo almacenar un tipo de dato. En este caso, todos los valores son considerados texto. Para mantener nuestros tipos de dato originales, debemos usar un data.frame

mi_data <- data.frame(students, mi_matrix)
mi_data

# Now, to assign names to the columns, we use the function 'colnames'
colnames(mi_data)<- c("Name", "age", "height", "grade", "GPA", "weight")

Funciones

Como ya has visto, podemos invocar funciones existentes como mean o cbind para hacer ciertas tareas. Hay muchas funciones en R, y cada vez que quieras hacer algo, debes comenzar por buscar si ya existe alguna función para ésto.

For instance, to get a vectors with 20 random numbers normally distributed: 
randNumbers <- rnorm(20)
randNumbers

Actividad

Encuentra la función que te permita encontrar el valor mínimo en randNumbers

Creando mis funciones

Si no encontramos funciones que nos sirvan, podemos crear las nuestras. En R, funciona similar a la creación de variables en el sentido que podemos asignar una función con’<-’.

La siguiente función multiplica a y b, y divide el resultado entre c

myFunction <- function(a,b,c)
{
  x <- (a*b)
  y <- x/c
  y
}

# We can invoke this function using the following parameters a=10 b=5, c=2 like
myFunction(10,5,2)
# Likewise
myFunction(b=5, c=2, a=10)

La función Apply en R

La función apply es a veces compleja de entender, pero puede ser muy útil. Las funciones apply ejecutan un conjunto de operaciones (una función dada) para todas las filas de un conjunto de datos.

Explora: ??apply

# For instance, if we create a function to print numbers from a list one by one:
printNumber <- function(number)
{
  textToPrint <- paste('The number is: ',number)
  print(textToPrint)
}

printNumber(3)
printNumber(5)

# We can invoke this function for all the elements from randNumbers 
output <- lapply(randNumbers, printNumber)

Nota que le pasamos el nombre de nuestra función (printNumber) como un parámetro a lapply. Cuál es el resultado?
