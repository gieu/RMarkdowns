Variables

El símbolo <- asigna un valor a una variable Por ejemplo, si queremos asignar el valor 5 a la variable ‘y’, hacemos:

y <- 5
# Y si queremos ver el valor contenido de la variable la ponemos sola:
y

Ahora podemos usar la variable y. En el siguiente bloque, multiplicamos y por 2, y luego lo asignamos a la variable x.

y*2
# Como no asignamos la operación a ninguna variable, y tendrá el mismo valor.
y
# Ahora guardaremos el resultado en otra variable llamada 'x':
x <- y*2
# Veamos el valor de 'x'
x

Vectores

Ahora vamos a trabajar con otro tipo de variables. Vamos a crear un listado de datos llamado vector. Un vector es una lista de datos almacenada en una única variable. Para crear un vector, usamos la instrucción c(): 
z <- c(1.1, 9, pi, 5 , 6, 7, 8) 
# ¿Qué significa tener 'pi' dentro del vector?

# Miremos el valor de la variable z
z

Este vector puede ser usados para crear otros más grandes. Por ejemplo, si queremos crear un vector con todos los valores en z, pero que tambén incluya los valores ‘x’ y ‘y’, hacemos lo siguiente:

c(x,z,y)

Como podemos ver, x tenía un valor de 10 y y tenía un valor 5, al utilizar el comando c() nos unío esos valores a la lista z en su respectivo orden.

Conceptos fundamentales

Vamos a definir algunos conceptos básicos y fundamentales a los cuales haremos referencia continuamente:

Unidad estadística, Individuo o Elemento: Personas u objetos que contienen cierta información que se desea estudiar y que pertenecen a la población en estudio.

Población: Conjunto de individuos o elementos que cumplen ciertas propiedades y entre los cuales se desea estudiar un determinado fenómeno.

Muestra: Subconjunto representativo de una población.

Variables* o caracteres: Característica observable que varía entre los diferentes individuos de una población. Las variables pueden dividirse en cualitativas y cuantitativas.

Tipos de Variable

Variables cualitativas: Se usan con datos que representan categorías que son mutuamente excluyentes, aunque se utilicen números para cada categoría no representan cantidades. Para su medición usamos escalas:

  • Nominales: No hay relación entre las categorías Una escala nominal, se lo permite clasificar (no jerarquizar ni ordenar). Ejemplo; Género, nacionalidad, ubicación Geográfica.

  • Ordinales: Sus posibles categorías se encuentran jerarquizadas y ordenadas. Ejemplo; Satisfacción de un usuario, grado de dolor.

Variables cuantitativas: Se usan con datos que se expresan mediante cantidades numéricas que permiten hacer operaciones matemáticas. Existen los siguientes tipos:

  • Discretas: Solo puede tomar valores enteros. Ejemplo; Edad, n° de hijos, días del año.

  • Continuas: Si sus posibles valores están en un conjunto infinito. Las podemos categorizar en intervalos Ejemplo; Peso, estatura, presión arterial.

Medidas de tendencia central

Las medidas de tendencia central son medidas estadísticas que pretenden resumir en un solo valor a un conjunto de valores. Representan un centro en torno al cual se encuentra ubicado el conjunto de los datos. Las medidas de tendencia central más utilizadas son: media, mediana y moda

  • Media: La media aritmética es un promedio estándar que a menudo se denomina promedio. Es el promedio de un conjunto de valores, o su distribución.
valores <- c(3,2,3,3,4,2,4)
mean(valores)

valores <- c(3,2,3,3,4,2,4)
median(valores)

Nota que en los datos tenemos una columa que describe el tipo y número de cursos previos: PriorCourses. Si queremos comparar el campo de Pretest a Posttest seún diferentes experiencias, podemos comenzar por ver la media y la desviación estándar para cada grupo utilizando la función aggregate
